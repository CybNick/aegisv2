# Purpose

Define how AI capabilities are integrated into Aegis while preserving determinism, explainability, and user trust.

AI is an assistant layer.

AI is never the source of truth.

The Knowledge Graph remains the authoritative source.

---

# Design Principles

Human Oversight

Explainability

Evidence-Based Outputs

Deterministic Core

Confidence Transparency

No Autonomous Actions
