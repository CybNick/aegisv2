# Engineering Standards

## Code Standards

* Type hints required
* Linting mandatory
* No dead code

---

## Testing Standards

### Unit Coverage

Minimum:

80%

Target:

90%

---

### Integration Coverage

Required for:

* APIs
* Connectors
* Storage

---

## Documentation Standards

Every module must include:

* Purpose
* Inputs
* Outputs
* Examples

---

## Security Standards

* Secrets never hardcoded
* Principle of least privilege
* Secure defaults

---

## UI Standards

* Accessibility compliant
* Responsive
* Keyboard accessible
