# Continuous Monitoring

Version: 1.0

---

## Purpose

Maintain an up-to-date understanding of infrastructure.

---

## Monitoring Functions

### Discovery

Detect:

* New assets
* New services
* New identities
* New relationships

---

### Change Detection

Detect:

* Configuration changes
* Permission changes
* Exposure changes
* Dependency changes

---

### Risk Updates

Automatically:

* Recalculate scores
* Update criticality
* Refresh analysis

---

## Monitoring Cycle

1. Collect observations

2. Normalize

3. Generate events

4. Update graph

5. Detect drift

6. Recalculate analysis

7. Generate notifications

---

## Drift Categories

### Added

Previously absent.

---

### Removed

Previously present.

---

### Modified

Changed state.

---
