# Aegis Executive Trust Audit

## Objective
Verify the integrity of every executive metric displayed in the Executive Mode dashboard. For Aegis to be trusted by leadership, no metric can be derived from opaque "AI black-boxes."

## Metric Verification

### 1. Security Score
- **Formula**: `(Total Assets - Critical Risk Assets) / Total Assets * 100` (Bounded to 0-100)
- **Inputs**: Absolute count of all live nodes vs count of nodes flagged with severity > 7.0 by the Exposure Engine.
- **Evidence Trail**: Each "Critical Risk Asset" can be clicked to view the exact `EXPOSURE` or `COMPROMISE` edges that triggered the flag.
- **Explainability**: 100% Deterministic.

### 2. Compliance Score
- **Formula**: `(Passed Controls / Total Evaluated Controls) * 100`
- **Inputs**: The `ComplianceEngine` iterates through the `mappings.py` rule dictionary against the live GraphView properties.
- **Evidence Trail**: A failed control (e.g. `CIS 4.1`) outputs the specific Node IDs that lack the required property (e.g. `port=22` exposed).
- **Explainability**: 100% Deterministic.

### 3. Open Recommendations
- **Formula**: Count of active recommendation objects generated by the `RecommendationEngine`.
- **Inputs**: Recommendation rules evaluated against the current `as_of` graph state.
- **Evidence Trail**: Recommendations are tightly coupled to the exact node subgraphs that triggered them. If the underlying nodes are patched (and the subsequent Scan Connector append confirms it), the recommendation deterministically vanishes.
- **Explainability**: 100% Deterministic.

### 4. Ownership Coverage
- **Formula**: `Assets with OWNED_BY edge / Total Assets * 100`
- **Inputs**: The `LifecycleEngine` enumerates nodes looking for explicit incoming ownership relationships.
- **Evidence Trail**: Navigating to the "Orphaned Assets" list provides the exact hardware/software nodes that are lowering this score.
- **Explainability**: 100% Deterministic.

## Conclusion
**GO** - Every executive-level metric is derived from basic arithmetic operations executed over a deterministically populated graph. There are zero heuristic estimations, ML inferences, or generative hallucinations involved in scoring. Aegis is entirely fact-based.
