# User Personas

## Security Analyst

Needs:

* Exposure visibility
* Asset awareness
* Risk prioritization

## Security Engineer

Needs:

* Infrastructure relationships
* Technical evidence
* Change tracking

## SOC Analyst

Needs:

* Monitoring
* Alerts
* Historical comparison

## Platform Engineer

Needs:

* Service dependencies
* Configuration understanding
* Operational visibility

## Enterprise Architect

Needs:

* System mapping
* Dependency understanding
* Impact analysis

## Security Leadership

Needs:

* Executive reporting
* Trends
* Risk summaries

## Beginner Users

Needs:

* Guided workflows
* Simple dashboards
* Easy-to-understand reporting
