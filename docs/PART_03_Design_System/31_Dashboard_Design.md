# Dashboard Design

Version: 1.0

---

# Philosophy

The dashboard is not a landing page.

It is an operational command center.

Every element must answer:

"What should I know right now?"

---

# Layout

Top Summary Row

Health Row

Analysis Row

Timeline Row

Insights Row

# Row 1 — Executive Metrics

Total Assets

Externally Reachable Assets

Critical Assets

Security Score

# Row 2 — Environment Health

Charts:

Asset Growth

Risk Trend

Change Activity

---

# Row 3 — Operational Intelligence

Panels:

Most Critical Assets

Highest Risk Systems

Recently Changed Assets

Validation Summary

---

# Row 4 — Timeline

Examples:

# Row 5 — Intelligence Panels

Dependency Summary

Cloud Coverage

---

# Right Side Panel

Always Visible

Contains:

Alerts

Monitoring Status

Scheduled Jobs

Recent Reports

---

# Dashboard Behavior

Updates automatically.

No manual refresh required.

---

# Empty Dashboard State

When no data exists:

Show guided onboarding.

Actions:

1. Create Workspace

2. Configure Scan

3. Run Discovery

4. Review Results

---

# Dashboard Goal

Provide complete situational awareness in under 30 seconds.
