# Accessibility Standards

Version: 1.0

---

# Goal

Aegis must be usable by all users.

---

# Compliance Target

WCAG 2.2 AA

Minimum

---

# Requirements

Keyboard Navigation

Screen Reader Support

High Contrast

Focus Indicators

Reduced Motion

---

# Keyboard Navigation

All actions accessible without a mouse.

---

# Focus States

Visible

Consistent

Never removed.

---

# Color Accessibility

Information must never rely solely on color.

Always include:

Labels

Icons

Text

---

# Typography Accessibility

Minimum body size:

14px

Preferred:

16px

---

# Motion Accessibility

Respect:

prefers-reduced-motion

---

# Screen Reader Support

All:

Buttons

Inputs

Charts

Graphs

must provide labels.

---

# Testing

Accessibility checks required before release.

---

# Goal

Every user can successfully operate the platform.
