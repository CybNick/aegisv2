# Iconography System

Version: 1.0

---

# Purpose

Icons provide rapid visual recognition of objects, actions, status, and relationships.

Icons must support:

* Fast scanning
* Visual consistency
* Enterprise readability

---

# Icon Style

Use:

Outlined Icons

Characteristics:

* 2px stroke
* Rounded corners
* Minimal detail
* Consistent geometry

---

# Icon Sources

Recommended:

* Lucide
* Heroicons
* Custom Security Icons

Avoid:

* Emoji
* Filled icon sets
* Mixed styles

# Design Rules

Never use icon-only actions unless obvious.

Icons must always align to text baselines.

Maximum icon size:

24px

Standard:

16px

20px

---

# Goal

Icons should improve comprehension without becoming decorative.
